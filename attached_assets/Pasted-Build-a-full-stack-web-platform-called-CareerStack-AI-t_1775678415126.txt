Build a full-stack web platform called **CareerStack AI** that helps fresh graduates continuously build, track, and showcase their practical project portfolios and measure job-readiness based on current market demand.

This is a production-grade MVP, so build it as a scalable web application with clean architecture.

==================================================
CORE PROBLEM TO SOLVE
==================================================

Fresh graduates currently lack a structured and credible way to continuously build, track, and showcase practical work aligned with real job market demands.

Their work is fragmented across multiple platforms, lacks standardization, and is not mapped to relevant job roles or skills.

This platform should bridge the gap between:
- learning
- skill validation
- portfolio building
- job readiness
- employment opportunities

--------------------------------------------------
2. PORTFOLIO BUILDER
--------------------------------------------------

Create a structured portfolio submission form.

Students should be able to add multiple projects.

Each project must contain:

- project title
- description
- problem solved
- technologies used
- difficulty level
- github link
- live deployment link
- screenshots
- duration
- role in project
- project category
- completion status

Support multiple project submissions.

Store projects in normalized DB tables.

--------------------------------------------------
3. TECH STACK COMFORT SCORE ENGINE
--------------------------------------------------

Create a scoring engine that analyzes all projects submitted.

The platform should calculate:

- most frequently used tech stack
- confidence score per technology
- comfort score per technology
- practical experience score

Example:

Python used in 5 projects = high confidence

React used in 1 project = medium confidence

Formula example:

comfort_score = 
(number_of_projects_using_tech × project_complexity × completion_score)

Normalize score out of 100.

Show as:

- Python → 88
- React → 72
- SQL → 64

Display visually using progress bars and radar chart.

--------------------------------------------------
4. MARKET DEMAND ENGINE
--------------------------------------------------

Build a demand engine that maps current job market demand.

For every tech stack, fetch:
- total open jobs
- trend score
- growth demand
- required skills

Example:

Python demand → 92
React demand → 85
Java demand → 74

Use scraped jobs data for this.

--------------------------------------------------
5. JOB READINESS SCORE
--------------------------------------------------

Generate a dynamic job-readiness score.

Formula should combine:

- comfort score
- roadmap completion
- market demand score
- portfolio strength
- skill gap analysis

Example formula:

job_readiness_score =
(comfort_score * 0.35) +
(market_demand * 0.25) +
(roadmap_completion * 0.25) +
(portfolio_quality * 0.15)

Final score out of 100.

Display:
- score card
- readiness status
- suggestions

Example:

78 → Interview Ready
54 → Needs Improvement
